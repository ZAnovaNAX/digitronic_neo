# DIGITRONIC - Автоматизированный сайт (Astro + Decap CMS)

Добро пожаловать в проект сайта DIGITRONIC. Сайт построен на быстром современном фреймворке **Astro**, со стилями на базе **Tailwind CSS**, и использует **Decap CMS** для удобного управления контентом, аналогично Tilda.

## Технологический стек

* **Astro**: Генератор статических сайтов (молниеносная скорость).
* **Tailwind CSS**: Мощный инструмент для верстки и стилизации по вашему дизайну.
* **React**: Вспомогательная библиотека для компонентов.
* **Decap CMS**: Система управления контентом (CMS). Работает прямо с файлами (Markdown) и имеет панель администратора для визуального редактирования.

## Как запустить проект локально для разработки (Гайд для MacBook Intel)

Если вы используете MacBook на базе процессоров Intel, выполните следующие шаги:

### 0. Подготовка окружения (если еще не установлено)

Вам потребуется установленный Node.js. Самый простой способ установить его на Mac - через Homebrew.

Откройте приложение **Терминал (Terminal)** и выполните:
```bash
# Установка Homebrew (если его еще нет)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Установка Node.js
brew install node
```

### 1. Установка зависимостей проекта

Перейдите в папку с проектом в терминале (используйте команду `cd путь/к/папке/проекта`) и выполните:
```bash
npm install
```

### 2. Запуск локального сервера CMS (Терминал 1)

Decap CMS требуется локальный прокси-сервер, чтобы сохранять изменения напрямую в локальные файлы. В терминале выполните:

```bash
npx decap-server
```

*(Не закрывайте этот терминал! Он должен работать в фоновом режиме)*

### 3. Запуск основного сайта (Терминал 2)

Откройте **новое окно терминала** (нажмите `Cmd + T` или `Cmd + N`), также перейдите в папку с проектом и выполните:

```bash
npm run dev
```

### 4. Использование

* **Сайт**: Перейдите в браузере по адресу `http://localhost:4321`
* **Панель управления (CMS)**: Перейдите по адресу `http://localhost:4321/admin/`

## Редактирование через CMS

Перейдя по `http://localhost:4321/admin/`, вы увидите интерфейс Decap CMS. Вы можете редактировать:
* **Страницы**: Главная, О компании, Программное обеспечение (тексты, картинки, списки).
* **Каталог товаров**: Удобное добавление/редактирование комплектов, загрузка изображений, изменение цены и категорий.

При сохранении изменений в админке, CMS автоматически изменит/создаст файлы в папке `src/content/` в вашем проекте!

## Развертывание (Деплой) в продакшн

Когда вы будете готовы опубликовать сайт:

1. Измените в файле `public/admin/config.yml` параметр `local_backend: true` на `# local_backend: true` (закомментируйте).
2. Запушьте код на GitHub или GitLab.
3. Подключите репозиторий к Netlify или Vercel (рекомендуется).
   * **Команда сборки (Build command)**: `npm run build`
   * **Директория (Publish directory)**: `dist`
4. Для Netlify: включите Netlify Identity, чтобы пользователи могли авторизоваться в `/admin/`.

## Итог

Дизайн был полностью скопирован из ваших HTML-макетов (Stitch), разбит на компоненты и переведен под управление контентом. Теперь вы можете настраивать всё как в Tilda через `localhost:4321/admin/`.
